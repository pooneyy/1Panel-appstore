## 简介

Percona Server for MySQL 是任何 MySQL 数据库的免费、完全兼容、增强且开源的直接替代品。它提供卓越的性能、可扩展性和仪器。

Percona Server for MySQL 受到数千家企业的信赖，可以为其最苛刻的工作负载提供更好的性能和并发性。
它通过优化的性能、更高的性能可扩展性和可用性、增强的备份以及更高的可见性为 MySQL 服务器用户提供更高的价值。
