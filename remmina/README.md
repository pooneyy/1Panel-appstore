# Remmina

Remmina 是一款开源、跨平台的远程桌面客户端，支持多种协议和Web界面。

## 功能特性

- 支持RDP、VNC、SPICE、NX、XDMCP等多种远程桌面协议
- 支持Web界面管理（默认端口3000）
- 支持多平台和多架构
- 支持SSH隧道和多种认证方式
- 轻量级、资源占用低

## 使用方法

1. 部署后访问 `http://服务器IP:3000` 进入Web管理界面
2. 添加远程主机信息，选择协议进行连接
3. 配置文件目录：`./config`
4. 数据目录：`./data`
