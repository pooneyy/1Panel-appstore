# Synapse Admin

Synapse 是一个开源的 Matrix 家庭服务器实现，由 Element 开发和维护。Matrix 是安全且可互操作实时通信的开放标准。

## 安装说明

本应用为 Synapse 服务器的管理工具，用于管理 Synapse 服务器的配置文件，用户，房间等信息。
