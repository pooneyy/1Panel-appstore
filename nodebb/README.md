## 简介

《NodeBB》是一个下一代讨论平台，它利用 WebSockets 实现即时交互和实时通知。
NodeBB 论坛自带许多现代功能，如社交网络集成和流式讨论，以及完整的 REST API 和插件框架，以实现第三方完全扩展。
