## 特性

+ 清爽界面，强大功能，低资源消耗
+ 🍊 易用，可视化操作，零代码使用
+ 一键切换内外网模式
+ 支持 Docker 部署（兼容 Arm 系统）
+ 🎪 支持多账户隔离
+ 🎏 支持查看系统状态
+ 🫙 支持自定义 JS、CSS
+ 🍻 无需连接外部数据库的简单用法
+ 🍾 免费组合丰富图标样式，支持 Iconify 图标库
+ 支持在网页中打开小窗口（某些第三方网站可能阻止此功能）
