# JellyFin

**Jellyfin** 是一个免费开源的媒体服务器软件，用于组织、管理和流媒体共享您的音频、视频和图片等媒体内容。

## 安装说明

+ 开启 `投屏服务(DLNA)` 与 `网络唤醒服务(WOL)` 功能

开启后，可以在局域网内的设备上投屏观看视频。 需要选择主机网络(host)模式。

## 中文字幕支持 (启用备用字体)

请在设置中添加备用字体路径
默认路径：`/home/jellyfin/config/font`

`/home/jellyfin` 为安装根路径，请在应用参数中查询具体参数
